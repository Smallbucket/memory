## 技术收集整理

### [vim](https://github.com/TourDJ/memory/blob/master/vim.md)    
Vim是一个高度可配置的文本编辑器，用于有效地创建和更改任何类型的文本。 它包含在大多数UNIX系统和Apple OS X的“vi”中。

Vim岩石稳定，并且不断发展以变得更好。 其功能包括：
* 持久的，多级撤消树
* 广泛的插件系统
* 支持数百种编程语言和文件格式
* 强大的搜索和替换
* 与许多工具集成
